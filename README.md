# Лабораторная работа №8: Адаптивная верстка с CSS Grid

## Цель работы
Освоить современные технологии CSS-разметки — Grid для создания адаптивных веб-страниц.

## Выполненные задания

### 1. Базовые примеры Grid
- Создание Grid-контейнера
- Работа с grid-template-columns и grid-template-rows
- Использование единицы измерения fr
- Именованные области Grid
- Выравнивание элементов

### 2. Практические примеры
- Простая сетка 3x3
- Макет страницы с использованием grid-template-areas
- Адаптивная галерея изображений

### 3. Самостоятельное задание
Создание Grid-макета дашборда с:
- Header (на всю ширину, 80px)
- Sidebar (левая колонка, 200px)
- Main content (основная область)
- Widget 1 и Widget 2 (правая колонка, 300px × 200px каждый)
- Footer (на всю ширину, 60px)

## Сравнение Flexbox и CSS Grid

| Критерий | Flexbox | CSS Grid |
|----------|---------|----------|
| Тип компоновки | Одномерная | Двумерная |
| Управление | Строка или столбец | Строки и столбцы |
| Основное назначение | Выравнивание элементов | Построение макетов |
| Подходит для | Компонентов | Страниц и сеток |

**Вывод:** Flexbox лучше подходит для линейных структур, Grid — для табличных и сложных.

## Типичные сценарии использования

| Задача | Рекомендуется |
|--------|--------------|
| Навигационное меню | Flexbox |
| Карточки товаров | Flexbox + Grid |
| Центрирование элемента | Flexbox |
| Макет страницы | Grid |
| Галерея изображений | Grid |
| Дашборд | Grid |
